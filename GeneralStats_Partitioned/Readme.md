# GeneralStats_Partitioned
The procedure GeneralStats_Partitioned is a procedure that will process a set of data to provide a range of statistics.

Each Partition will be evaluated as a group of data. The Partition can be passed as a composite key in your provided dataset, allowing for more flexibility to know what information to group together to evaluate and run the calculations against, including Dates and aggregated Dates. That means that if you provide 3 different groupings in your dataset, the Box And Whisker data will be the same for each Grouping respectively across the 3 different groupings (where each grouping would have a different set of distinct results).

Inputs:
* ExternalID (Provided ID field from inputs, this will be pulled from the specified table)
* ExternalCode (Provided Code field from inputs, this will be pulled from the specified table)
* Value (Provided Value field from inputs, this will be pulled from the specified table. This is the Value used for calculations)
* IndependantValue (Provided IndependantValue field from inputs, this will be pulled from the specified table. This is the Value used for calculations)
* PartionGroup (Provided Grouping/Partition field from inputs, this will be pulled from the specified table. This is the Range or Grouping of Values used for calculations)
* ErrorPercentage

Outputs:
* ID (Unique Identifier Created each run)
* ExternalID (Provided ID field from inputs, this will be pulled from the specified table)
* ExternalCode (Provided Code field from inputs, this will be pulled from the specified table)
* Value (Provided Value field from inputs, this will be pulled from the specified table. This is the Value used for calculations)
* IndependantValue (Provided IndependantValue field from inputs, this will be pulled from the specified table. This is the Value used for calculations)
* PartionGroup (Provided Grouping/Partition field from inputs, this will be pulled from the specified table. This is the Range or Grouping of Values used for calculations)
* COUNT_Value (Calculated COUNT of VALUES per PartionGroup)
* SUM_Value (Calculated TOTAL SUM of all VALUES per PartionGroup)
* AVG_Value (Calculated AVERAGE of all VALUES per PartionGroup)
* STDEV_Value (Calculated STANDARD DEVIATION of all VALUES per PartionGroup)
* MIN_Value (Calculated MINIMUM VALUE per PartionGroup)
* Median_Value_Cont (Calculated Continuous MEDIAN of all VALUES per PartionGroup)
* Median_Value_Disc (Calculated Discrete MEDIAN of all VALUES per PartionGroup)
* MAX_Value (Calculated MAXIMUM VALUE per PartionGroup)
* COUNT_IndependantValue (Calculated COUNT of INDEPENDENT VALUES per PartionGroup)
* SUM_IndependantValue (Calculated TOTAL SUM of all INDEPENDENT VALUES per PartionGroup)
* AVG_IndependantValue (Calculated AVERAGE of all INDEPENDENT VALUES per PartionGroup)
* STDEV_IndependantValue (Calculated STANDARD DEVIATION of all INDEPENDENT VALUES per PartionGroup)
* MIN_IndependantValue (Calculated MINIMUM INDEPENDENT VALUE per PartionGroup)
* Median_IndependantValue_Cont (Calculated Continuous MEDIAN of all INDEPENDENT VALUES per PartionGroup)
* Median_IndependantValue_Disc (Calculated Discrete MEDIAN of all INDEPENDENT VALUES per PartionGroup)
* MAX_IndependantValue (Calculated MAXIMUM INDEPENDENT VALUE per PartionGroup)
* Slope 
* Pearson_R
* Pearson_R_Sqrd
* DegOfFreedom
* ConfidenceLevel
* CriticalValue
* t_value
